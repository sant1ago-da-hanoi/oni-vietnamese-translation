# Quy ước dịch — Oxygen Not Included Tiếng Việt

Cập nhật: 2026-04-30

## Nguyên tắc chung

- **Giọng văn**: thân mật, hài hước, gần gũi — đúng tinh thần game
- **Không dịch**: tên riêng (Duplicant names), mã format (`{0}`, `{Skills}`), tag HTML/Unity (`<link>`, `<style>`, `<b>`)
- **Nhất quán**: mỗi thuật ngữ chỉ dùng MỘT cách dịch xuyên suốt
- **Ngắn gọn**: ưu tiên từ ngắn cho UI (nút bấm, label), có thể dài hơn cho mô tả

## Quy tắc đặc biệt

### 1. Duplicant = đệ
- Viết thường: **đệ** (chỉ viết hoa ở đầu câu)
- Có thể dùng "thằng đệ" trong ngữ cảnh thân mật / hài hước
- Số nhiều: "các đệ", "mấy đứa đệ"
- Sở hữu: "của đệ", "cho đệ"

### 2. Giữ nguyên tiếng Anh khi
- Tên riêng sinh vật: Hatch, Morb, Puft, Drecko, Pip, Shove Vole...
- Tên nguyên tố hóa học đặc thù game: Abyssalite, Isoresin, Fullerene...
- Thuật ngữ kỹ thuật quen thuộc: Stress, DNA, pH...
- Tên DLC: Spaced Out!, The Frosty Planet Pack

### 3. Format string
- **KHÔNG** dịch hoặc thay đổi: `{0}`, `{1}`, `{Skills}`, `{DeadDupe}`...
- **KHÔNG** dịch hoặc thay đổi tag: `<link=\"...\">`, `<style=\"...\">`, `<b>`, `</b>`
- Giữ nguyên `\n` (xuống dòng) đúng vị trí

### 4. Viết hoa
- Đầu câu
- Tên riêng (tên nhân vật, tên DLC)
- KHÔNG viết hoa giữa câu cho thuật ngữ game (đệ, thuộc địa, tinh thần...)
- KHÔNG viết hoa toàn bộ trừ khi bản gốc viết hoa toàn bộ

### 5. Giọng văn theo ngữ cảnh
- **UI label / nút bấm**: ngắn gọn, trung tính — "Xây", "Đào", "Nghiên cứu"
- **Mô tả building**: rõ ràng, đầy đủ thông tin
- **Thông báo / tooltip**: có thể hài hước nhẹ, đúng tinh thần game
- **Codex / Lore**: nghiêm túc hơn, văn phong kể chuyện
