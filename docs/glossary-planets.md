# Thuật ngữ — Hành tinh & Tiểu hành tinh

## Quy tắc đặt tên

- **Asteroid** → dùng "tinh cầu" (mặc định) hoặc "tiểu hành tinh" — cần thống nhất 1 từ
- **Cluster** → "cụm" + tên hành tinh
- **Moonlet** → "tiểu nguyệt"
- **Fragment** → "mảnh"
- **Minor** → "nhỏ" (hậu tố)
- **Blasted** → "tan vỡ"
- Tên riêng hành tinh (Marinea, Ceres, Relica) → dịch sang tiếng Việt, KHÔNG giữ English
