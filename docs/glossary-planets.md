# Thuật ngữ — Hành tinh & Tiểu hành tinh

Cập nhật: 2026-06-14

## Quy tắc đặt tên

| Yếu tố | Dịch | Ghi chú |
|--------|------|---------|
| Tên hành tinh (worlds) | X tinh (3-4 âm) | Chỉ áp dụng cho hành tinh có tên riêng trong game |
| Cluster | Cụm + tên | |
| Moonlet | tiểu nguyệt | |
| Fragment | Mảnh + tên | |
| Minor | Tiểu + tên | VD: Tiểu hải dương tinh |
| Blasted | + tan vỡ (hậu tố) | |
| Space destinations | X tinh (thống nhất) | Dùng cùng pattern với worlds |

**Lưu ý:**
- Tất cả tên hành tinh riêng (worlds/clusters/space destinations) giờ đều dịch sang Hán Việt, không giữ English.
- Pattern "X tinh" áp dụng cho TẤT CẢ — worlds, asteroids, space destinations.
