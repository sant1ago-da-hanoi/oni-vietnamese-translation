## Quy tắc đặt tên

| Yếu tố | Dịch | Ghi chú |
|--------|------|---------|
| Tên hành tinh | X tinh (3-4 âm) | Thống nhất toàn bộ. Bỏ "tinh cầu" |
| Cluster | Cụm + tên | |
| Moonlet | tiểu nguyệt | |
| Fragment | Mảnh + tên | |
| Minor | Tiểu + tên | VD: Tiểu hải dương tinh |
| Blasted | + tan vỡ (hậu tố) | |
| Asteroid (biến thể nhỏ) | giữ tên gốc hoặc thêm mô tả | |

**Lưu ý:** Tất cả tên hành tinh riêng giờ đều dịch sang Hán Việt, không giữ English.
