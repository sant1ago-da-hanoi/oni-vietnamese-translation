# Nguyên tắc chung

- **Giọng văn**: thân mật, hài hước, gần gũi — đúng tinh thần game
- **Không dịch**: tên riêng (Duplicant names), mã format (`{0}`, `{Skills}`), tag HTML/Unity (`<link>`, `<style>`, `<b>`)
- **Nhất quán**: mỗi thuật ngữ chỉ dùng MỘT cách dịch xuyên suốt
- **Ngắn gọn**: ưu tiên từ ngắn cho UI (nút bấm, label), có thể dài hơn cho mô tả
